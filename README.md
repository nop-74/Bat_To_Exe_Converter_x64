# Bat_To_Exe_Converter_x64
Bat To Exe Converter - удобная программа для быстрой конвертации .BAT
- Bat To Exe Converter - удобная программа для быстрой конвертации .BAT файлов в исполняемый формат .zip. В архиве есть портативная версия, интерфейс - поддерживает Русский язык, при установке не забываем снимать лишние галочки
![Alt text](https://github.com/nop-74/Bat_To_Exe_Converter_x64/blob/main/1.png)

- Главное окно программы имеет в себе все настройки, никуда больше переключаться нет необходимости. Перед тем как конвертировать файл, вы сможете - настроить параметры видимости, указать рабочий каталог, можете указать, что делать с временными файлами, если хотите можете воспользоваться - - шифрованием, можно сжать .zip файл с помощью UPX, указать архитектуру. В целом это все, удачной работы, надеюсь программа будет вам полезна

Команда shutdown используется в разных операционных системах для управления завершением работы системы. winitpro.ruen.wikipedia.orgSkyeng.ru
В Windows
Shutdown — встроенная команда командной строки, которая позволяет выключить, перезагрузить компьютер, перевести его в спящий режим или завершить сеанс пользователя. winitpro.rucomputerra.ru
Некоторые параметры команды:
•	/s — полное отключение системы.
•	/r — перезапуск компьютера.
•	/h — переход в режим гибернации.
•	/l — завершение пользовательского сеанса.
•	/f — принудительное закрытие программ.
•	/a — отмена любых запланированных операций.
 kod.ru
Пример использования для настройки отложенного отключения: kod.ru
1.	Нажать сочетание клавиш Win + R для вызова диалогового окна «Выполнить».
2.	Ввести команду 
shutdown /s /t XXXX
, где XXXX — период до отключения в секундах.
3.	Подтвердить выполнение нажатием Enter или кнопки «ОК».
 kod.ru
Важно: в Windows команды shutdown обязательно используют слеши (/), а не дефисы (-). kod.ru
В Linux
Shutdown обеспечивает контролируемое завершение работы системы. Синтаксис: 
shutdown [OPTIONS] [TIME] [MESSAGE]
. Skyeng.rulinuxize.com
Основные опции:
•	-h — выключение системы (halt);
•	-r — перезагрузка системы (reboot);
•	-c — отмена запланированного выключения;
•	-k — только отправка уведомления, без реального выключения.
 Skyeng.ru
Примеры использования:
•	sudo shutdown -h now
 — немедленное выключение;
•	sudo shutdown -h +5
 — выключение через 5 минут;
•	sudo shutdown -r 23:30
 — перезагрузка в 23:30.
 Skyeng.ru
Для выполнения команды потребуются права администратора. Если работа идёт не под учётной записью root, перед командой нужно добавить 
sudo
. Skyeng.ru
Важно: в современных дистрибутивах Linux (особенно в тех, что используют systemd) команда shutdown обычно является символической ссылкой на 
systemctl
. Skyeng.ru
