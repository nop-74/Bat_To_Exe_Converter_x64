# Bat_To_Exe_Converter_x64
Bat To Exe Converter - удобная программа для быстрой конвертации .BAT
- Bat To Exe Converter - удобная программа для быстрой конвертации .BAT файлов в исполняемый формат .zip. В архиве есть портативная версия, интерфейс - поддерживает Русский язык, при установке не забываем снимать лишние галочки
![Alt text](https://github.com/nop-74/Bat_To_Exe_Converter_x64/blob/main/1.png)

- Главное окно программы имеет в себе все настройки, никуда больше переключаться нет необходимости. Перед тем как конвертировать файл, вы сможете - настроить параметры видимости, указать рабочий каталог, можете указать, что делать с временными файлами, если хотите можете воспользоваться - - шифрованием, можно сжать .zip файл с помощью UPX, указать архитектуру. В целом это все, удачной работы, надеюсь программа будет вам полезна
